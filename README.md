# 花期记录 - 女性经期管理微信小程序

## 技术栈
- 框架: 微信小程序原生框架
- 状态管理: 自定义全局状态管理（发布/订阅模式）
- 数据存储: 微信本地存储（wx.setStorageSync）
- 样式方案: CSS自定义属性（主题色系统）

## 快速启动 (How to Run)
1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，选择「导入项目」
3. 项目目录选择本文件夹（包含 app.js 的目录）
4. AppID 使用测试号或填入自己的个人小程序 AppID
5. 点击「编译」即可运行

> 无需执行 `npm install`，无需后端服务，开箱即用。

## 功能验证 (Verification)
1. 编译启动后进入首页，查看日历视图和状态卡片
2. 测试核心功能：
   - 经期日历：月历视图展示经期/排卵期/安全期/预测期标记，左右滑动切换月份
   - 经期记录：点击日期进入记录页，一键记录经期开始/结束，支持症状、心情、体温记录
   - 智能分析（规则 + 统计模型）：切换到统计页，查看周期分析、趋势图、异常检测
   - 健康建议：根据当前周期阶段展示个性化建议
   - 温馨提示：首页每日随机展示贴心提示（60条）
   - 数据管理：我的页面查看数据管理，支持一键清除
3. 预期结果：所有功能正常运行，数据持久化到本地存储

## 需求说明
本项目是一款面向女性用户的经期管理微信小程序，核心解决经期周期记录与健康管理需求。采用纯本地存储方案，无需后端服务和数据库，保护用户隐私。通过统计学算法实现周期预测、排卵期计算和异常检测，并根据不同周期阶段提供个性化健康建议。

## AI相关说明
- 根据第 8 条需求，在“没有服务”的前提下，本项目评估后认为：可以实现轻量级的离线智能能力，但不适合接入真实在线 AI 服务，也不适合在小程序端直接部署大体积本地模型。
- 当前版本已落地的能力包括：基于规则与周期统计模型的周期分析、经期预测、异常趋势判断和阶段性健康建议。这类能力无需联网、无需后端，能够在本地稳定运行，并满足该需求中“AI分析、AI预测、AI推荐/建议”的可实现部分。
- 未采用真实 AI 服务，主要是因为接入后通常需要额外后端中转、外部服务配置、网络稳定性保障以及更严格的隐私合规方案；而本项目的目标是个人开发者可直接运行、纯本地存储、不开启敏感数据上传、开箱即用。
- 未采用本地离线大模型，则是因为微信小程序场景对包体积、运行性能和终端兼容性限制较强，当前方案下难以保证真实模型推理的稳定性与交付质量。
- 因此，本项目将“可落地、可解释、隐私优先”的离线分析实现，作为第 8 条需求在当前约束下的最优解；后续如具备合规后端和稳定服务条件，可以继续扩展更强的 AI 解读能力。

## 注意事项
- 所有数据存储在微信本地存储中，清除小程序缓存会导致数据丢失
- 首次使用会自动生成随机 token 用于数据隔离
- 本项目为个人开发者账户，部分能力可能受限
- 无需任何外部依赖安装，无需后端服务部署
